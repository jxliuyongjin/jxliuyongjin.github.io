一、MySQL8的安装

  1. 在 root 目录下，安装 mysql 和 mysql-devel
   yum install mysql
   yum install mysql-devel
  1. 安装mysql-server
wget http://dev.mysql.com/get/mysql80-community-release-el7-5.noarch.rpm
rpm -ivh mysql80-community-release-el7-5.noarch.rpm
yum install mysql-community-server
  1. 重启mysql服务
service mysqld start
  1. 查看 Mysql 运行状态
service mysqld status

![](https://atts.w3cschool.cn/attachments/image/20230807/1691393979642658.png)
  1. 查看初始密码
cat /var/log/mysqld.log|grep localhost

  1. 进入mysql
mysql -uroot -p
  1. 修改密码
alter user 'root'@'localhost' identified by 'Abc123456!';
  1. 退出
exit;

二、修改远程访问权限

  1. 登录mysql
mysql -uroot -p
  1. 查看mysql库中的user表host字段
使用mysql库：
use mysql
select user,host from user;
可以看到root后面的host是localhost表示只允许本机访问，需要把它改成%，%表示允许任何host访问

![](https://atts.w3cschool.cn/attachments/image/20230807/1691394458607245.png)
  1. 修改root用户的host字段，命令
update user set host ="%" where user ="root"；

再次查看user表host字段 这里root的host字段变成%了

  1. 使用本次修改立即生效，命令：
flush privileges;

  1. 登录腾讯云，找到云服务器

  2. 开放mysql对应的接口

  3. 完成，可以用Navicat远程连接了
