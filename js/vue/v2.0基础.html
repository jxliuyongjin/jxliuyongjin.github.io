# 三、基础

# 1、VUE指令

vue会根据不同的【指令】，针对标签实现不同的【功能】

指令：带有v-前缀的 特殊 标签属性

# 1.1、v-html

插入一段html代码

# 1.2、v-show,v-if

前者隐藏，通过display:none控制隐藏，后者直接不创建。

# 1.3、v-if,v-else,v-else-if

两者中间不了能有别的标签，要紧挨着

# 1.4、v-on

a、v-on:事件名="表达式"

<button v-on:click="count++"><button>

b、v-on:事件名="函数名称"

c、@标签代替v-on:

<button @click="count++"><button>

# 1.5、v-bind

# a、基础用法

插值不能用在标签属性中，v-bind:就提供了相应的功能

<img v-bind:src="url">

简写方式
<img :src="url">

# b、v-bind对class的操作

语法：:class ="对象/数组"

1、通过控制对象的布尔值来实现某个class是否起作用
<div class="box" :class="{class1:ture,class2:false}"></div>

2、通过设置数组，使得数组中所有累，都会添加到盒子桑，本质就是一个class列表
可以通过修改数组来控制样式的作用
<div class="box" :class="[class1,class2,class3]"></div>

3、直接修改行内样式,这种方式很方便直接修改某一个属性
<div class="box" :style="{width:"20px",height:"20px","backgrround-color":"green"}"

# 1.6、v-for

v-for="(item,index) in 数组"

<ul>
  <li v-for="(item,index) in list">
  	{{item}}--{{index}}
  </li>
</ul>
或者
<ul>
<li v-for="item in list">
  {{item}}
</li>
</ul>

////////for中的Key
<ul>
<li v-for="item in list" :key="item.id">
  {{item.name}}
</li>
</ul>
key没有key,for会默认原地修改元素，（就地复用）

# 5、自定义指令

# 5.1、全局注册

在main.js中添加如下代码

<input v-focus ref="inp" type="text">
Vue.directive('focus', {
  // inserted 会在 指令所在的元素，被插入到页面中时触发
  inserted (el) {
    // el 就是指令所绑定的元素
    // console.log(el);
    el.focus()
  }
})

# 5.2、局部注册

<input v-focus type="text">
<script> 
  // 2. 局部注册指令
  directives: {
    // 指令名：指令的配置项
    focus: {
      inserted (el) {
        el.focus()
      }
    }
  }
}
</script>

# 5.3、值设置

<template>
  <div>
    <h1 v-color="color1">指令的值1测试</h1>
    <h1 v-color="color2">指令的值2测试</h1>
  </div>
</template>

<script>
export default {
  data () {
    return {
      color1: 'red',
      color2: 'orange'
    }
  },
  directives: {
    color: {
      // 1. inserted 提供的是元素被添加到页面中时的逻辑
      inserted (el, binding) {
        // console.log(el, binding.value);
        // binding.value 就是指令的值
        el.style.color = binding.value
      },
      // 2. update 指令的值修改的时候触发，提供值变化后，dom更新的逻辑
      update (el, binding) {
        console.log('指令的值修改了');
        el.style.color = binding.value
      }
    }
  }
}
</script>
