6、行内元素和块级元素

行内元素：不独自占一行的是行内元素（input,span)

  • 行内元素只能写行内元素，不能写块级元素。不然容易出现错乱

块级元素：独自占一行的是块级元素（h，p,div)

  • 块级元素能写行内元素和块级元素
  • p标签中不能写块元素

# 标签、

# 1、文本标签

  • 用于包裹：词汇、短语等。
  • 通常写在排版标签里面(h元素,p,div)。
  • 排版标签更宏观（大段的文字），文本标签更微观（词汇、短语）。
  • 文本标签都是行内元素。

# 1.1、常用

标签名标签语义单/双标签
em要着重阅读的内容双
strong十分重要的内容（语气比em强）双
span没有语义，用于包裹短语的通用容器双

# 1.2、不常用

标签名标签语义单/双标签
cite作品标题（书籍，歌曲，电影等）双
dfn特殊术语，或者专属名词双
del与ins删除的文本【与】插入的文本双
sub与sup下标文字【与】上标文字双
code一段代码双
samp从正常的上下文中，将某些内容提取出来，比如：标识设备的输出双
kbd键盘文本，表示文本时通过键盘输入的，通常用于与计算机相关的手册中双
abbr缩写，最好配合上title(缩写含义的说明）属性双
bdo文本倒序显示，要配合dir属性，可选值：ltr(默认）,rtl双
var标记变量，可以和code标签一起使用双
small附属细则，例如：包括版权，法律文本。--很少使用
b摘要的关键字，评论中的产品名称。--很少使用双
i呈现字体图标双
u与正常内容有反差文本，比如错误的单词，不合适的描述等。--很少使用双
q短引用--很少使用双
blockquote长引用--很少使用双
address地址信息双

注：blockquote和address是块级元素，其他事行内元素

# 2、列表

# 2.1、ol：有序列表

<ol>
    <li> 第1步 </li>
    <li> 第2步 </li>
    <li> 第3步 </li> 
</ol>

# 2.2、ul：无序列表

<ul>
  <li> 第1步 </li>
  <li> 第2步 </li>
  <li> 第3步 </li> 
</ul>

# 2.3、dl:自定义列表

<dl>
    <!-- dt标题 -->
    <dt> 标题1 </dt>
    <!-- 数据 -->
    <dd> 数据1 </dd>
    <dt> 标题2 </dt>
    <dd> 数据2 </dd>
    <dt> 标题3 </dt>
    <dd> 数据3 </dd>
</dl>

# 2.4、注意事项

  • 无论ol还是ul，里面子元素尽量全是li标签，如果需要添加别元素最好外面包裹li

    <ol>
    <li> 第1步 </li>
    <li> 第2步 </li>
    <li> 第3步 </li> 
    <li> <a herf= "www.baidu.com">百度</a></li>
</ol>

  • 单个li标签尽量不要用

  • 列表可以互相嵌套

    <ol>
  <li> 第1步 </li>
  <ul>
      <li> 第1步 </li>
      <li> 第2步 </li>
      <li> 第3步 </li> 
    </ul>
  <li> 第3步 </li>  
</ol>

  • 自定义dl里面的一个dt可以多个dd

    <dl>
    <!-- dt标题 -->
    <dt> 标题1 </dt>
    <!-- 数据 -->
    <dd> 数据1 </dd>
    <dd> 数据2 </dd> 
</dl>

# 3、table 表格

# 3.1、基本格式

  • 基本结构：标题，头部，主体，脚注(可选)组成
  • tr表示一行
  • th标签出现再头不
  • td标签出现再主体和脚注
  • caption里面没有属性
<!-- border 默认单位事像素 -->
<table border ="1" width="500" height="200" cellspacing="0" >
    <!-- 标题 -->
    <caption>学生信息</caption> 
    <!-- 头部 -->
    <thead>
        <!-- 行 tr，table row -->
        <tr>
            <!-- th,tbale heard -->
            <!-- 跨越2列 -->
            <th>项目</th>
            <th>名字</th>
            <th>性别</th>
            <th>年龄</th>
            <th colspan="3">任课</th>
        </tr>
    </thead>
    <!-- 主体 -->
    <tbody align="center"> 
        <!-- 行 -->
        <tr>
            <!-- td，table body -->
            <td rowspan="2"> 老师 </td>
            <td> 张三 </td>
            <td> 男 </td>
            <td> 18 </td>  
            <td> 物理 </td> 
            <td> 数学 </td> 
            <td> 体育 </td>  
        </tr>
        <tr>
            <!-- td，table body -->
            <td> 李四 </td>
            <td> 女 </td>
            <td> 28 </td>  
            <td> 语文 </td> 
            <td> 英语 </td> 
            <td> 音乐 </td> 
        </tr>
    </tbody>
    <!-- 脚主(可选) -->
    <tfoot align="right">
        <tr>
            <!--可以和上面不同的列数 --> 

            <td colspan="7">共计：2人</td> 
        </tr>
    </tfoot>
</table>

# 3.2、常用属性

# 3.2.1、table的属性

1、border

<!-- border 默认单位事像素 -->
<table border='10'>

注：如果border的值大于1，只能表示最外层边框的粗细，内部边框依旧是1，需要调节内部边框的大小可以通过css控制

2、width

每一列的宽度默认是根据里面的内容自动调节

3、height

表示表哥的最低高度，调节height调节高度只会改变主体内每一行的高度，头部和脚注的高度不会改变

4、cellspacing

边框之间的距离，当值为0时，边框不会消失，至少两个边框的距离时0

# 3.2.2、thead的属性

1、height

当这个height值大于table的height 值时，table表格会自动拉长让height保持高度

2、align

水平对齐方式有，left,center,right,justify,char方式，具体查看文档

3、valign

垂直对齐方式，有bottom，middle，top，baseline等方式

# 3.2.3、tbody的属性

1、height

这个受限与table中的高度，这个值得满足table总体的高度的最小值，否则就不生效

2、其他属性

align、align、valign等属性同thead的属性

# 3.3.1、th的属性

1、colspan rowspan

<!-- 跨越2列 -->
<th colspan="2">项目</th>
<!-- 跨越2行 -->
 <td rowspan="2"> 老师 </td>

# 4、form 表单

# 4.1、作用

主要是用来交互的，收集用户的信息

# 4.2、格式

<form action = "https://www.baidu.com/s" target="_self" method="get">
    <!-- 文本输入 -->
    账号：<input type = "text" name = "accout" value ="zhansan" maxlength="10">
    <!-- 密码输入 -->
    密码：<input type = "password" name = "pwd"   maxlength="6"> <br>
    <!-- 单选 -->
    性别：
    <input type="radio" name="sex" value="male">男
    <input type="radio" name="sex" value="female">女<br>
    <!-- 复选框 -->
    爱好：
    <input type="checkbox" name="hobby" value="hejiu" checked>喝酒 <br>
    <input type="checkbox" name="hobby" value="chouyan" >抽烟
    <input type="checkbox" name="hobby" value="tangtou" >烫头 <br>
    <br>
    <!-- 隐藏域 -->
    <input type="hidden" name="tag" value="123">
    <br>
    <!-- 确认按钮，方式1 -->
    <button>确定</button>
    <!-- 确认按钮，方式2 不能写name属性-->
    <input type ="submit" value ="确认"> <br>
    <!-- 重置按钮，方式1 -->
    <button type ="reset">重置</button>
    <!-- 重置按钮，方式2 不能写name属性-->
    <input type ="reset" value ="重置"> <br>
    <!-- button 默认type ="submit"
        由此想要做普通按钮需要设定按钮类型为button
    -->
    <button type="button">检车是否已注册</button>
    <input type ="button" value ="检车是否已注册"> <br>
</form>
  • 发出的请求：https://www.baidu.com/s?wd=手机
  • action:请求地址
  • input的name作为请求的值对，需要和后台确认
