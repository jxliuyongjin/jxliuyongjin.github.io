# 一、下载与安装

# 1、下载地址

https://www.google.cn/chrome/

下载完成安装

# 2、迁移chrome安装目录

  • 在谷歌浏览器快捷方式上点击鼠标右键，打开“属性”

  • 在“属性”里找到“起始位置”，可以看到默认安装在c盘，复制。并粘贴到我的电脑中打开谷歌浏览器安装位置;

  • 将C盘下谷歌浏览器安装目录中所有文件剪切到要想要安装的盘里(E:\Program Files\chrome);

  • 点击右键创建新的谷歌浏览器快捷方式至桌面即可。

# 二、缓存目录迁移出C盘

# 1、查看chrome配置

chrome://version/

修订版本	46fa58c804a9efde819900e8cee01821bde4576f-refs/branch-heads/5845@{#1610}
操作系统	Windows 10 Version 1909 (Build 18363.1443)
JavaScript	V8 11.6.189.19
用户代理	Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/116.0.0.0 Safari/537.36
命令行	"E:\Program Files\chrome\chrome.exe" --flag-switches-begin --flag-switches-end hao.360.cn/?src=lm&ls=n4b58e79ca3
可执行文件路径	E:\Program Files\chrome\chrome.exe
个人资料路径	C:\Users\Administrator\AppData\Local\Google\Chrome\User Data\Default
使用中的变体	50eaf59c-377be55a

个人资料路径就是缓存目录

# 2、移除缓存数据

  • 关闭chrome浏览器
  • 剪切C:\Users\Administrator\AppData\Local\Google\Chrome\User Data 文件夹到你想要的文件夹（E:\Program Files\chrome）

# 3、建立连接

进入cmd，执行命令mklink /j命令：

mklink /J "C:\Users\Administrator\AppData\Local\Google\Chrome\User Data" "E:\Program Files\chromedata\User Data"

输出结果为

为 C:\Users\Administrator\AppData\Local\Google\Chrome\User Data <<===>> E:\Program Files\chromedata\User Data 创建的联接

这样你就能在 C:\Users\Administrator\AppData\Local\Google\Chrome\User Data 目录下看到生成一个带小箭头的 User Data 图标了。

就是欺骗欺骗浏览器的作用，不占用空间，而真正的个人资料就在刚刚移动的 e 盘中了
