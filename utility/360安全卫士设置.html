# 一、设置

360安全卫士太多弹窗广告，需要修改设置

# 1、找到设置

打开安全卫士->右上角落三横->设置

# 2、基础设置

# 2.1、关闭功能定制

找到基础设置，找到功能定制，全部取消

# 2.2、关闭用户体验改善计划

找到户体验改善计划，取消

# 2.3、关闭云安全计划

找到云安全计划-->>加入云安全计划，发现可以文件后自动上报，取消

# 3、弹出设置

# 3.1、关掉wifi安全提示

# 3.2、关掉用户账号及任务，关掉

# 3.3、关掉用户账号及任务，关掉

# 3.4关闭产品推荐

找到弹窗设置，找到产品推荐提醒，全部取消

# 3.4、青少年保护模式，关掉

# 3.5、游戏优化结果，关掉

# 3.6、开机今日精选提醒，关掉

# 3.7、开启轻松一下提醒，关掉

# 3.8、个性化推荐，关掉

# 3.9、开启每日热点，关掉

# 3.10、开机今日精选提醒，关掉

# 3.11、安全云定制服务，关掉

# 3.12、开启今日趣看提醒，关掉

# 3.13、开机咨询推荐，关掉

# 4、关闭开机小助手

找到开机小助手，全部取消

# 5、木马查杀

找到内容推荐。关闭
